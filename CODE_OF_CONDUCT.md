# Code of Conduct

## Our pledge

We want this to be a welcoming, low-friction place to do public-data work.
That requires basic respect between people, and basic standards for
discourse.

## Expected behavior

- Assume good faith in PRs and issues.
- Critique ideas, not people.
- Be specific when you disagree. "This is wrong" is unhelpful; "row 42
  has the wrong field name, upstream says `station_id` not `Station Id`"
  is helpful.
- Welcome newcomers. The first PR is the hardest.
- Give credit where it's due.

## Unacceptable behavior

- Harassment, personal attacks, or bigotry of any kind.
- Publishing others' private information.
- Bad-faith engagement (drive-by criticism, deliberate obstruction).
- Undisclosed conflicts of interest in analyses (financial, political,
  employer-related).

## Enforcement

Project maintainers will:
- Remove comments, commits, or PRs that violate this code.
- Temporarily or permanently ban contributors for repeated or severe
  violations.
- Explain decisions when asked, privately if necessary.
