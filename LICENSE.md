## Data: per-dataset upstream license

The catalog's `license` column records each dataset's license as published
by its upstream portal. Common values for our sources:

- **OGL-Ontario** — Open Government Licence – Ontario (data.ontario.ca)
- **ODC-BY** — Open Data Commons Attribution Licence (open.toronto.ca)
- **CC-BY** — Creative Commons Attribution
- **CC0** — Public domain dedication
- **custom** — see `raw/SOURCE.md` for details

**By contributing a dataset folder, you confirm** that the data's license
permits the kind of redistribution you're doing (typically: snapshot +
analysis), or that you're only committing derived/transformed data and
metadata, not the upstream file itself.

## Attribution

When you reuse data from this repo, you must:

1. Attribute the original publisher (cited in `raw/SOURCE.md`).
2. Indicate any changes you made.
3. Not imply the publisher endorses your analysis.

This applies whether or not the upstream license formally requires it. It's
also just good practice.
